# Processed Data Directory

This directory contains cleaned and processed data.

Files in this directory are generated by running:
```bash
python scripts/data_processing.py
```

## Contents

- Individual processed datasets from each source
- Merged dataset combining all sources
- Derived metrics and calculations

## Data Processing Steps

1. Cleaning (missing values, duplicates)
2. Standardization (country names, formats)
3. Merging (combining multiple sources)
4. Calculation (derived metrics)
