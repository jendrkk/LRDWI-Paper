# Raw Data Directory

This directory contains raw data downloaded from various sources.

Files in this directory are generated by running:
```bash
python scripts/data_scraper.py
```

## Data Sources

- World Bank API
- OECD Statistics
- World Inequality Database (WID.world)

## Note

Large data files may be excluded from git tracking via .gitignore.
See the main README.md for instructions on downloading the data.
