# Output Figures

This directory contains all generated figures and plots for the paper.

Figures are generated by running the Jupyter notebooks:
- `notebooks/01_data_analysis.ipynb`
- `notebooks/02_visualizations.ipynb`

## Figure Types

- Time series plots
- Cross-country comparisons
- Distribution visualizations
- Correlation heatmaps
- Multi-panel figures

All figures are saved in high resolution (300 DPI) PNG format suitable for publication.
